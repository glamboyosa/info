# Info Coding Challenge

## Run the app

To run the app, first install dependencies via:

```bash
npm install
#or
yarn
```

Then run:

```bash
npm run dev
#or
yarn dev
```

This will start a server on `http://localhost:3000`

## Optional - Code Formatting

To format your code with prettier run:

```bash
npm run prettier
#or
yarn prettier
```
